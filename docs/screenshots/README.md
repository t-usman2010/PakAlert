## ðŸ“ Screenshot Guidelines

### Technical Specs
- **Resolution:** 1920x1080 (Full HD) minimum
- **Format:** PNG for quality, JPG for smaller size
- **Browser:** Use Chrome/Edge for best rendering
- **Zoom:** 100% (default browser zoom)

### Best Practices
- Clean browser (no bookmarks bar, close unnecessary tabs)
- Full screen or consistent window size
- Hide personal information
- Ensure good contrast and readability
- Use actual data (not dummy data if possible)

### Taking Screenshots

**Windows:**
- `Win + Shift + S` â†’ Select area
- `Win + Print Screen` â†’ Full screen to Pictures folder
- Use Snipping Tool or Snip & Sketch

**macOS:**
- `Cmd + Shift + 4` â†’ Select area
- `Cmd + Shift + 3` â†’ Full screen

**Browser Extensions:**
- Awesome Screenshot
- Nimbus Screenshot
- FireShot
